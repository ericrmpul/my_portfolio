# 娛樂為王-用 RQI 量化 YouTube 內容的變現品質
 
> 從一位重度使用者的觀看紀錄出發，設計一套能把「觀看紀錄」轉換成「變現品質訊號」的指標，再做成可持續的預警機制。
 
**分析師**：Chun Yao Wang
**資料**：個人 YouTube 觀看紀錄（103 天、10,464 筆觀看）
**工具**：SQL（資料清理）· Power BI（計算邏輯 + 視覺化）· KPI 框架設計
**完整報告**：[YouTube_RQI_Analysis.pdf](./YouTube_RQI_Analysis.pdf)
 
---
 
## 問題
 
一位重度使用者，每天看 247 分鐘 YouTube，其中娛樂加短片佔了 72.81%。表面上這只是「看很多」，但我想問一個對平台有意義的問題：
 
> 當低變現價值的內容（娛樂、短片）長期主導一個人的觀看結構，這對平台的長期變現能力代表什麼？而我們能不能把這件事量化、甚至持續監控？
 
這份分析以單一重度使用者為案例，重點不在「證明平台怎麼了」，而在示範一套**從行為資料設計變現品質指標、並轉成預警機制**的方法——這套方法本身才是可以擴展到平台規模的東西。
 
## RQI：把內容消費轉成一個變現訊號
 
我依不同內容類型的廣告變現潛力（參考各類型 CPM 區間）給定權重——投資/科普 ×4、運動/音樂 ×2、娛樂/短片 ×1——設計出 RQI（Revenue Quality Index）：
 
```
RQI = (高價值內容加權分 / 全部內容加權分) × 100%
```
 
RQI 不是真實營收數字，而是一個方向性的評分模型，用來呈現不同觀看組合在「變現品質」上的結構差異。以這位使用者的資料計算，RQI = 33.07，遠低於我設定的警戒線 60，代表其內容消費結構存在明顯的變現效率落差。
 
## 主要發現
 
- **低價值內容結構性主導：** 娛樂與短片壟斷逾七成觀看量，且短片集中在通勤、午休、睡前等過渡時段，比較像碎片時間的慣性填補，而非主動選擇。
- **從一次性診斷到持續預警：** 加上閾值（RQI < 60）後，這個指標就從「算一次看結果」變成可以每週追蹤的監控訊號。當低價值消費下降、RQI 回升越過閾值時，機制會如預期反應——展示的是預警邏輯能運作，不是模型已被實證驗證。
- **行為分群定位：** 用 CPM 高低 × 觀看時長兩軸，把使用者放進四象限。此案例落在高參與、低變現潛力的一群，這正是平台最難變現、卻佔流量最大的類型。
## 決策方向
 
報告最後從數據收斂出三個平台層級的行動方向：依內容類型差異化短片的變現門檻以吸引高 CPM 創作者；在短片觀看越過閾值時介入推送高價值內容；以及依使用者歷史標籤，在高參與內容中插入跨類別的高價值廣告。
 
## 已知限制
 
- **單一使用者資料**，無法直接推論到平台整體；報告中平台規模的數字是條件性推估，不是實測。
- **RQI 是假設性模型**，權重來自對 CPM 差異的假設，而非實證迴歸。
- **四象限的分布比例**為概念性示意，沒有實測用戶資料支撐。
